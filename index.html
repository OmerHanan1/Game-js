About the project!

This work has been submitted by Amit and Omer, it uses JQuery and does not use a template.

The project was really enjoyable for us,
as we both work as full-stack developers.
The most challenging part was making the game work smoothly without built-in states, as in React.js, so we created state objects and helpers functions, and we are really proud of it.

We have learned a lot about the client-side fundamentals.
